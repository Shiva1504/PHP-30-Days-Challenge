# Day 10: Superglobals Basics

## 📌 Topics Covered

* `$_GET` → Capture query string values
* `$_POST` → Capture form submission values
* `$_SERVER` → Access server and request info

## 📝 Exercise

1. Create a simple HTML form inside PHP.
2. Use `$_GET` to capture and print query parameters.
3. Use `$_POST` to capture and print form input.
4. Print basic `$_SERVER` details.

## ✅ Example Output

```
Hello, John! (from GET)
Hello, Shiva! (from POST)

Server Info:
Request Method: GET
Server Name: localhost
PHP Self: /Day-10/solution.php
```
